# Paste Image

MarkDown 编辑时粘贴图片处理

## 常用变量

| 变量                          | 名称           |
| ----------------------------- | -------------- |
| \${currentFileName}           | 当前文件名     |
| \${currentFIleNameWithoutExt} | 当前文件名     |
| \${currentFileDir}            | 当前文件夹目录 |
| \${projectRoot}               | 项目跟路径     |
